# Snack Shop - Ecommerce App

A simple and modern React-based ecommerce application for browsing and purchasing snacks.

## Features

- Browse a selection of snack products with images
- Add products to shopping cart
- Update quantities in cart
- Remove items from cart
- View total price

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
